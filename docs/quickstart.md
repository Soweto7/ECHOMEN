# Quickstart

## 1. Install
```bash
npm install
```

## 2. Configure
Create `.env`:
```bash
API_KEY=your_google_api_key_here
```

## 3. Run
```bash
npm run dev -- --host 0.0.0.0 --port 4173
```

Open `http://localhost:4173`.

## 4. First run
Use prompt:
> "Plan and execute a one-page launch brief with action items."

Then open **Artifacts** and click **Share Run** to export a sanitized markdown execution report.
