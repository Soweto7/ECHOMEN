# Demo Flows (Short Recordings)

## 1) Hero Workflow (90s)
- **Narrative:** "From idea to finished artifact in one run."
- **Prompt:** "Draft a launch post, generate a changelog entry, and produce a markdown brief for my team."
- **Capture checklist:**
  1. Task plan appears.
  2. Multi-step execution starts.
  3. Final artifacts panel shows output.
  4. Use **Share Run** and show generated report.

## 2) Local Cheap Stack (60s)
- **Narrative:** "Run ECHO with local models for minimal cost."
- **Prompt:** "Analyze this draft and suggest 3 copy edits."
- **Capture checklist:**
  1. Open model provider config.
  2. Select local/Ollama endpoint.
  3. Execute lightweight task.
  4. Show token/cost-friendly workflow.

## 3) Multi-Agent Execution (90s)
- **Narrative:** "Planner + Executor + Reviewer collaboration."
- **Prompt:** "Create a release summary, QA checklist, and rollback plan."
- **Capture checklist:**
  1. Planner decomposes tasks.
  2. Executor runs tools.
  3. Reviewer pass/fail loop.
  4. Synthesized final report artifact.

## Recording spec
- Resolution: 1440x900 or 1920x1080.
- Aspect options: 16:9 for YouTube, 9:16 crop for shorts.
- Always include: task pipeline, live logs, artifacts panel.
